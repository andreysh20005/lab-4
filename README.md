# Задание 1

## Задача 1

### Текст задачи

Дерево содержит вещественные числа. Отбросить в каждом из них дробную
часть.

### Алгоритм решения

формируем список случайной длинны, состоящий из вещественных значений (от -100.0 до +100.0), далее, заносим элементы в дерево с помощью insert. Получаем бинарное дерево поиска. Далее, создаём новое дерево из имеющегося. Для этого обходим дерево (из-за чегов новом будут отсортированы элементы) и переводим в int, оставляя только целую часть. Для обхода используется treeFold, применяющий функцию int к каждому обробатываемому элементу

## Задача 2

### Текст задачи

Сформировать список из узлов с одним листом (узел является листом, если у
него нет ни левого, ни правого поддерева).

### Алгоритм решения

формируем список случайной из случайных строк. Он заносится в дерево, получается бинарное дерево поиска. Далее, oneLeafNode() обходит каждую вершину и сравнивает с образцами на то, является ли дочерние вершины листяьями (одна вершина, эта вершина - лист). Данные вершины заносятся в лист, который выводится пользователю
